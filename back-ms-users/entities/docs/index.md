# User Management APIs

## Overview

This documentation covers two complementary APIs for comprehensive user and location management:

- **User API**: Complete user account management
- **Location API**: Geographic data and address management

## APIs Included

### User API
Handles user registration, authentication, profile management, and account operations.

### Location API  
Manages user addresses, geographic data hierarchy, and location validation.

## Getting Started

Both APIs use OpenAPI specifications and standard REST conventions. Authentication is required for most operations.
