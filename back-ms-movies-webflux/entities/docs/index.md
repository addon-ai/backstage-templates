# Movie API

## Overview

The Movie API provides comprehensive movie rental management capabilities including catalog management, rental transactions, and inventory tracking.

## Features

- **Movie Catalog Management**: CRUD operations for movies
- **Rental Transaction Processing**: Handle rental requests and returns
- **Inventory Tracking**: Real-time availability management
- **Late Fee Calculation**: Automated fee processing
- **User Rental History**: Track user rental status and history

## Authentication

This API uses standard authentication mechanisms. Refer to the OpenAPI specification for detailed authentication requirements.

## Rate Limiting

API calls are rate-limited to ensure service stability. Check response headers for current limits.
